1. GitHub Blocks Assignment:

This is a project to practice the CSS Box Model (like padding and margins) and Flexbox alignment using (justify-content)

2. What I Learned and Did:

- I used padding and margins to control the space around elements and containers
- I used (display: flex) to align items horizontally in different ways
- I gave all the images a fixed size ("height: 200px" and "width: 200px") so they all look consistent

3. How the Flexbox Rows Work:

for Row 1 (.row-one):This row has a blue background. I used "justify-content: space-around" to put equal space around the three images.
for Row 2 (.row-two): This row has a light green background. I used "justify-content: center" to put the two images in the middle, and used "gap: 50px" to create space between them.
for Row 3 (.row-three):This row has a pink background. I used "justify-content: space-between" to push the images to the left, middle, and right sides.
